---
title: "FastAPI"
description: "OpenTelemetry instrumentation for FastAPI — traces HTTP requests and responses automatically."
---

Provides automatic and manual instrumentation of FastAPI applications, tracing HTTP requests using the underlying ASGI middleware.

## Installation

```bash
pip install opentelemetry-instrumentation-fastapi
```

## Usage

### Auto-instrumentation

`opentelemetry-instrument` handles this automatically.

```bash
opentelemetry-instrument uvicorn main:app
```

### Manual instrumentation

<Tabs>
  <Tab title="Instrument existing app">
    ```python
    import fastapi
    from opentelemetry.instrumentation.fastapi import FastAPIInstrumentor

    app = fastapi.FastAPI()

    @app.get("/foobar")
    async def foobar():
        return {"message": "hello world"}

    FastAPIInstrumentor.instrument_app(app)
    ```
  </Tab>
  <Tab title="Instrument globally">
    ```python
    from opentelemetry.instrumentation.fastapi import FastAPIInstrumentor

    # Patches fastapi.FastAPI so all future instances are instrumented
    FastAPIInstrumentor().instrument()

    import fastapi
    app = fastapi.FastAPI()
    ```
  </Tab>
</Tabs>

## Configuration

<ParamField path="server_request_hook" type="callable">
  Called with the server span and the ASGI `scope` dict for every incoming request.
</ParamField>

<ParamField path="client_request_hook" type="callable">
  Called with the internal span, ASGI `scope`, and the ASGI `receive` event dict when `receive` is invoked.
</ParamField>

<ParamField path="client_response_hook" type="callable">
  Called with the internal span, ASGI `scope`, and the ASGI `send` event dict when `send` is invoked.
</ParamField>

<ParamField path="excluded_urls" type="string">
  Comma-delimited list of regexes matching URLs to exclude from tracing. Overrides `OTEL_PYTHON_FASTAPI_EXCLUDED_URLS`.
</ParamField>

<ParamField path="http_capture_headers_server_request" type="string[]">
  List of request header name regexes to capture as span attributes.
</ParamField>

<ParamField path="http_capture_headers_server_response" type="string[]">
  List of response header name regexes to capture as span attributes.
</ParamField>

<ParamField path="http_capture_headers_sanitize_fields" type="string[]">
  List of header name regexes whose values should be replaced with `[REDACTED]`.
</ParamField>

<ParamField path="exclude_spans" type="string[]">
  Optionally exclude `"receive"` and/or `"send"` child spans from the trace. Example: `["receive", "send"]`.
</ParamField>

<ParamField path="tracer_provider" type="TracerProvider">
  Optional tracer provider. Defaults to the globally configured provider.
</ParamField>

<ParamField path="meter_provider" type="MeterProvider">
  Optional meter provider. Defaults to the globally configured provider.
</ParamField>

### Environment variables

| Variable | Description |
|---|---|
| `OTEL_PYTHON_FASTAPI_EXCLUDED_URLS` | Comma-delimited URL regex exclusion list. |
| `OTEL_INSTRUMENTATION_HTTP_CAPTURE_HEADERS_SERVER_REQUEST` | Request header names (or regexes) to capture as span attributes. |
| `OTEL_INSTRUMENTATION_HTTP_CAPTURE_HEADERS_SERVER_RESPONSE` | Response header names (or regexes) to capture as span attributes. |
| `OTEL_INSTRUMENTATION_HTTP_CAPTURE_HEADERS_SANITIZE_FIELDS` | Header names whose values should be replaced with `[REDACTED]`. |

### Hooks example

```python
from opentelemetry.instrumentation.fastapi import FastAPIInstrumentor
from opentelemetry.trace import Span
from typing import Any

def server_request_hook(span: Span, scope: dict[str, Any]):
    if span and span.is_recording():
        span.set_attribute("custom.attribute", "value")

def client_request_hook(span: Span, scope: dict[str, Any], message: dict[str, Any]):
    if span and span.is_recording():
        span.set_attribute("asgi.receive.type", message.get("type", ""))

def client_response_hook(span: Span, scope: dict[str, Any], message: dict[str, Any]):
    if span and span.is_recording():
        span.set_attribute("asgi.send.type", message.get("type", ""))

FastAPIInstrumentor().instrument(
    server_request_hook=server_request_hook,
    client_request_hook=client_request_hook,
    client_response_hook=client_response_hook,
)
```

## Spans produced

A server span is created per HTTP request with kind `SERVER`. Child spans are created for each ASGI `receive` and `send` event (unless excluded via `exclude_spans`). The span name is set to the matched route pattern, e.g. `GET /users/{user_id}`. Standard attributes include:

- `http.method` / `http.request.method`
- `http.route`
- `http.status_code` / `http.response.status_code`
- `http.scheme`
- `http.host`
- `http.target`
- `http.user_agent`

## Supported versions

Requires `fastapi ~= 0.92`.
